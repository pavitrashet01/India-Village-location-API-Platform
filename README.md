# India-Village-Data-Analysis

Project Title
India Village Location API Platform

 Description
This project focuses on building a data pipeline and dashboard for analyzing village-level data across India. It involves data cleaning, database integration, and visualization.

Tech Stack
•	Python (Pandas) 
•	MySQL 
•	SQLAlchemy 
•	Power BI 

 Process
1. Data Collection
•	Combined 30 Excel datasets containing village data
3. Data Cleaning (ETL)
•	Removed null values and duplicates 
•	Dropped unnecessary columns 
•	Standardized column names 
•	Fixed data types 
4. Data Storage
•	Converted cleaned data into CSV 
•	Loaded ~400,000 records into MySQL database
6. SQL Analysis
•	Wrote queries for filtering, searching, and aggregation 
•	Implemented pagination for large data handling 
7. Visualization
•	Built Power BI dashboard showing: 
o	Total states, districts, villages 
o	Village distribution across states 
o	Top states by village count 
o	Geographic map visualization 

 Key Features
•	Efficient handling of large datasets 
•	Clean and structured database design 
•	API-ready SQL queries 
•	Interactive dashboard 

 Outcome
•	Built an end-to-end data pipeline 
•	Delivered insights using visualization 
•	Gained hands-on experience in real-world data engineering workflow
